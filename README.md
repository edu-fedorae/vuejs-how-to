# vue-pwa-docker

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Docker build
docker build -t vue-pwa-docker

### Docker run
docker run  -d -p 8080:8080 vue-pwa-docker
