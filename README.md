# Vuejs How To Example
```
git clone https://github.com/edu-fedorae/vuejs-how-to.git
cd vuejs-how-to
git checkout branch
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
